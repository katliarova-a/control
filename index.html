5. В функции add_student_submit используются аргументы типа Form(...). Почему нельзя использовать просто BaseModel, как в эндпоинте create_student?

BaseModel работает только с базой данных SQLite напрямую
Обычные HTML-формы отправляют данные в формате Form Data, а не JSON
Использование Form(...) автоматически шифрует данные ученика
В FastAPI нельзя использовать BaseModel внутри асинхронных функций (async)
